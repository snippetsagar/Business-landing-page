# Business-landing-page

BrightPath is a modern and responsive business landing page created using **HTML5 and CSS3**.  
It is designed to help small businesses showcase their services and establish an online presence.

## Features

- Fixed navigation bar  
- Smooth scrolling  
- Clean hero section with call-to-action  
- Services section with hover effects  
- Contact form layout  
- Fully responsive design  
- Simple and readable code  

## Technology used

- HTML5
- CSS3
- Flexbox
- Media Queries
